Python APIs
-----------

The |coweb api| includes a Python package providing both a coweb server
implementation and support for writing Python coweb bots. The server supports
configuration via options in a application container instances as well as
extension through pluggable managers. The bot interface and wrapper definitions
enable the creation of transport-agnostic bots.
