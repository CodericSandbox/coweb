Introduction
------------

The |coweb api| enables the creation of :term:`cooperative web application`\s featuring concurrent real-time interactions among users and external data sources. The framework handles remote notification of user changes, the resolution of conflicting changes, and convergence of application state using an operational transformation algorithm.

The following sections delve into cooperative web concepts and how they apply to the architecture and implementation of the coweb framework.
