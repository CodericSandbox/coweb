Getting started
---------------

The tutorials in this section describe how to start using the |coweb api|. The first explains how to install and deploy cooperative web applications on the reference Java and Python coweb servers. The second details the process of writing a cooperative web shopping list application, similar to the running example in the :doc:`/intro/openg` section.
